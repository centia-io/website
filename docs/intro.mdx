# Intro

Get up and running with Centia.io **in less than 5 minutes**.

## Getting Started

Get started by [sign up and create a new basebase back-end](https://api.centia.io)


## Install the CLI

<Tabs>
  <TabItem value="npm" label="npm" default>
```bash
npm install @centia-io/cli --global
```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
```bash
yarn global add @centia-io/cli
```
</TabItem>
  <TabItem value="pnpm" label="pnpm">
```bash
pnpm add @centia-io/cli --global
```
   </TabItem>
</Tabs>

## Use the CLI

Start by logging in to the back-end:
```bash
centia login
```

Create a new schema for your tables:
```bash
centia schema add "my_new_schema"
```

Create a new table in the schema:
```bash
centia table add "my_new_schema" "my_new_table"
```

Add columns in your table:
```bash
centia column add "my_new_schema" "my_new_table" "id" int
centia column add "my_new_schema" "my_new_table" "name" varchar
```

And maybe you want to add a primary key to your id column:
```bash
centia constraint add "my_new_schema" "my_new_table" "id" primary
```

So now your table is ready for insert of data:
```bash
echo "INSERT into my_new_schema.my_new_table values(1, 'Martin')" | centia sql
```

And run queries on your table:
```bash
echo "select * from my_new_schema.my_new_table" | centia sql
```
