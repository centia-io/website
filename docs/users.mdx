# Users

## Create a new user

<Tabs>
    <TabItem value="http" label="HTTP" default>

        ```http title="Create a new user"
        POST https://api.centia.io/api/v4/users HTTP/1.1
        Content-Type: application/json
        Authorization: Bearer abc123

        {
            "name": "joe",
            "email": "joe@example.com",
            "password": "Abc123!!"
        }
        ```

    </TabItem>
    <TabItem value="cli" label="CLI">

        ```bash title="Create a new user"
        cen user add "joe"
        ```

    </TabItem>
</Tabs>

## Get users

<Tabs>
    <TabItem value="http" label="HTTP" default>

        ```http title="A single user"
        GET https://api.centia.io/api/v4/users/joe HTTP/1.1
        Accept: application/json; charset=utf-8
        Authorization: Bearer abc123
        ```

    </TabItem>
    <TabItem value="cli" label="CLI">

        ```bash title="A single user"
        cen user get "joe"
        ```

    </TabItem>
</Tabs>

## Update user

<Tabs>
    <TabItem value="http" label="HTTP" default>

        ```http title="A single user"
        PATCH https://api.centia.io/api/v4/users/joe HTTP/1.1
        Accept: application/json; charset=utf-8
        Authorization: Bearer abc123

        {
            "password": "Def456!!"
        }

        ```

    </TabItem>
    <TabItem value="cli" label="CLI">

        ```bash title="A single user"
        cen user update "joe"
        ```

    </TabItem>
</Tabs>
