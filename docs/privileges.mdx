# Privileges

## Set privileges for user on table

<Tabs>
    <TabItem value="http" label="HTTP" default>

        ```http
        PATCH https://api.centia.io/api/v4/schemas/myschema/mytable/privilege HTTP/1.1
        Content-Type: application/json
        Authorization: Bearer abc123

        {
            "subuser": "joe",
            "privilege": "write"
        }
        ```

    </TabItem>
    <TabItem value="cli" label="CLI">

        ```bash
        cen privilege set "myschema" "mytable" "joe" "write"
        ```

    </TabItem>
</Tabs>

## Get privileges

<Tabs>
    <TabItem value="http" label="HTTP" default>

        ```http
        GET https://api.centia.io/api/v4/schemas/myschema/mytable/privilege HTTP/1.1
        Accept: application/json; charset=utf-8
        Authorization: Bearer abc123
        ```

    </TabItem>
    <TabItem value="cli" label="CLI">

        ```bash
        cen privilege set "myschema" "mytable"
        ```

    </TabItem>
</Tabs>
